BST rule: left side has smaller values, right side has bigger values.

Node* searchBST(Node *root, int key) {
    if (root == NULL || root->data == key)
        return root;

    if (key < root->data)
        return searchBST(root->left, key);

    return searchBST(root->right, key);
}
Node* insertBST(Node *root, int val) {
    if (root == NULL)
        return createNode(val);

    if (val < root->data)
        root->left = insertBST(root->left, val);
    else
        root->right = insertBST(root->right, val);

    return root;
}

4. Animated Lowest Common Ancestor

LCA is the lowest node that is ancestor of both target nodes.

Choose an LCA example.
Node* LCA(Node *root, Node *p, Node *q) {
    if (root == NULL || root == p || root == q)
        return root;

    Node *left = LCA(root->left, p, q);
    Node *right = LCA(root->right, p, q);

    if (left && right)
        return root;

    return left ? left : right;
}

5. Animated Height / Diameter

Diameter is the longest path between any two nodes. It may or may not pass through root.

Click animation.
int helper(Node *root, int *diameter) {
    if (root == NULL) return 0;

    int left = helper(root->left, diameter);
    int right = helper(root->right, diameter);

    *diameter = max(*diameter, left + right);
    return 1 + max(left, right);
}

10. Binary Tree Construction Patterns

FAANG often asks you to rebuild a tree from traversal arrays. This tests recursion, index mapping, and subtree boundaries.

Preorder + Inorder

Preorder gives root first. Inorder splits left and right subtree.

Postorder + Inorder

Postorder gives root last. Inorder again splits the tree.

Sorted Array to BST

Middle element becomes root to keep tree balanced.

// Build Balanced BST from sorted array
Node* buildBST(int arr[], int left, int right) {
    if (left > right) return NULL;

    int mid = left + (right - left) / 2;
    Node *root = createNode(arr[mid]);

    root->left = buildBST(arr, left, mid - 1);
    root->right = buildBST(arr, mid + 1, right);

    return root;
}

11. BST Delete - All 3 Cases

BST delete is one of the most misunderstood tree algorithms. Always remember the 3 cases.

Case 1: Leaf Node

Just delete it and return NULL.

Case 2: One Child

Return the existing child to connect it with parent.

Case 3: Two Children

Replace with inorder successor or predecessor.

Node* minValueNode(Node *root) {
    Node *curr = root;
    while (curr && curr->left != NULL)
        curr = curr->left;
    return curr;
}

Node* deleteBST(Node *root, int key) {
    if (root == NULL) return root;

    if (key < root->data)
        root->left = deleteBST(root->left, key);
    else if (key > root->data)
        root->right = deleteBST(root->right, key);
    else {
        // Case 1 and 2
        if (root->left == NULL) {
            Node *temp = root->right;
            free(root);
            return temp;
        }
        if (root->right == NULL) {
            Node *temp = root->left;
            free(root);
            return temp;
        }

        // Case 3: two children
        Node *succ = minValueNode(root->right);
        root->data = succ->data;
        root->right = deleteBST(root->right, succ->data);
    }
    return root;
}

12. Maximum Path Sum

This is a hard FAANG favorite. At every node, ask: what is the best path I can pass upward, and what is the best path using this node as the highest turning point?

Key Idea

Return only one side upward, but update global answer using left + root + right.

int maxPathHelper(Node *root, int *answer) {
    if (root == NULL) return 0;

    int left = maxPathHelper(root->left, answer);
    int right = maxPathHelper(root->right, answer);

    if (left < 0) left = 0;
    if (right < 0) right = 0;

    int throughRoot = root->data + left + right;
    if (throughRoot > *answer)
        *answer = throughRoot;

    return root->data + (left > right ? left : right);
}

13. Serialize / Deserialize Tree

Serialization converts a tree into a string. Deserialization rebuilds the same tree. This is used in distributed systems, storage, testing and interview hard problems.

Example Tree:
        1
       / \
      2   3

Preorder Serialization:
1,2,NULL,NULL,3,NULL,NULL

Why NULL matters?

Without NULL markers, many different trees can create the same traversal string.

Best Traversal

Preorder is common because root is processed before subtrees.

Complexity

Serialize O(n), deserialize O(n).
